# java-lab

A practical repository designed for Java developers who want to improve their understanding of algorithms, time complexity (Big O), and clean coding practices.  
Here you'll find real Java implementations and educational examples to support your learning journey.

-  Algorithm complexity (Big O)
-  Data structures
-  Programming best practices
-  Java API usage
-  Clean and readable code

---

##  Project Structure

```bash
java-lab/
├── src/
│   └── big_o_notation/
│       ├── Constant.java       # O(1)
│       ├── Logarithmic.java    # O(log n)
│       ├── Linear.java         # O(n)
│       ├── LinearLog.java      # O(n log n)
│       ├── Quadratic.java      # O(n²)
│       ├── Exponential.java    # O(2ⁿ)
│       ├── Factorial.java      # O(n!)
│       └── Main.java           # Main entry class

About the Project

This repository was created as a learning and experimentation lab with algorithms, data structures, and performance in Java.
Each example is designed with clarity, readability, and a focus on understanding time complexity.

How to Run

Clone the repository:

git clone https://github.com/mateusValentim6D76/java-lab.git


Open it in your preferred IDE (IntelliJ, Eclipse, VSCode...)

Navigate to the desired class and run the main method.

Requirements

Java 17 or higher

IDE or terminal with Java support
